💡 使用说明

由于GitHub Pages不支持Streamlit应用直接运行，请使用以下方式体验完整功能：

方法一：本地运行
1. 克隆仓库：git clone https://github.com/HeShen-1/LangChain.git
2. 安装依赖：pip install -r requirements.txt
3. 运行应用：streamlit run app.py

方法二：云端部署
推荐使用 Streamlit Cloud、Heroku、Railway 等平台部署

技术栈：Streamlit + LangChain + OpenAI GPT + FAISS + HuggingFace Embeddings
